# ぱるるワールドMobile

ブラウザで遊べるスゴロクゲームです。ボードをカスタマイズして、1人〜12人で遊べます。

## プレイ

**GitHub Pages:** https://mokablack.github.io/palulu-world-mobile/

`index.html` をブラウザで直接開いても動作します（ビルド不要）。

---

## ボードエディターの使い方

1. ページを開くと**ボードエディター**が表示されます
2. **グリッドサイズ**（行・列）を入力して「更新」を押します
3. **パレット**からタイルの種類を選び、ボードのマスをクリックして配置します
   - 進む・戻るマス：クリック後に移動マス数を入力します
   - 休みマス：クリック後に休みターン数を入力します
   - イベントマス：クリック後にイベントの種類を選択します
4. **アイテム**タブでゲーム中に登場するアイテムの有効/無効を設定します
5. **ステージ保存**ボタンでスロット（1〜3）に保存できます
6. **シングルプレイ**または**ローカルマルチ**タブでプレイヤーを設定してゲーム開始します

## オンラインマルチのFirebase設定

オンラインマルチプレイにはFirebase Realtime Databaseが必要です。

### 1. Firebaseプロジェクトの作成

1. [Firebase Console](https://console.firebase.google.com/) にアクセス
2. 「プロジェクトを追加」からプロジェクトを作成
3. 左メニューの「構築」→「Realtime Database」を開く
4. 「データベースを作成」→ロケーションを選択 → テストモードで開始

### 2. セキュリティルールの設定

Realtime DatabaseのルールタブにページTo内の「セキュリティルール例を表示」から取得したルールを貼り付けます。

### 3. APIキーとDatabase URLの取得

1. Firebase Console の「プロジェクトの設定」（歯車アイコン）を開く
2. 「全般」タブの「マイアプリ」→ ウェブアプリを登録（未登録の場合）
3. 「firebaseConfig」オブジェクトから以下をメモします：
   - `apiKey`
   - `projectId`
   - `databaseURL`

### 4. ゲームへの設定入力

1. エディター画面の「Firebase設定」フォームに `APIキー`・`プロジェクトID`・`Database URL` を入力
2. 「保存」を押すと設定がブラウザに保存されます
3. 「オンラインマルチ」タブからルームの作成または参加ができます
